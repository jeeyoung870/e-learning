[Deploying to codesandbox.io]

1) Import the project as is (do all necessary npm installs)

2) Set up dependencies: 
express
^4.17.1 (4.18.1)
pug
^3.0.2 (3.0.2)
ws
^7.5.3 (7.5.8)

External resources

3) Configure external resources: 
External URL
https://cdn.com/bootstrap.css 
Add Fonts:
Google Fonts 
Changes to 2 and 3 will be reflected automatically when modifying package.json.

4) Modify 'server.js' ports:
Replace
server.listen(3000, handleListen);
with
server.listen(process.env.PORT, handleListen); 

Replace
app.set("views", __dirname + "/public/views");
with
app.set("views", __dirname + "/views");
(As it recognizes "public" as the default path)

5) Run the project using codesandbox container or 'npm run dev'.
-> Success!!
